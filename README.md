### Dependencies

```sh
$ nvm install
$ npm install
```

### View the components in isolation locally

```sh
$ npm start
```

Navigate to the [running instance](http://localhost:6006) of
[storybook](https://storybook.js.org/)

### Verification

```sh
$ npm test
```
